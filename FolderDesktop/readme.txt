众所周知桌面只是一个普通的文件夹。但为什么桌面不可以是别的文件夹？
desktop.ini文件被用来设置一个文件夹的属性比如背景图片和图标。但是这个文件应用并不广泛，甚至不少人误以为是病毒。
现在Windows10支持多桌面（按Ctrl + Windows + D可以新建一个桌面，或者点击任务栏里的任务视图），为什么不可以在创建一个新桌面的时候指定另一个文件夹做为桌面？

As we all know, desktop is just a folder. But why can't desktop be another folder?
desktop.ini file is used to set a folder's attribute, like background image and icon. but seldom do people know this file, even some may think it's some kind of virus.
Now Windows 10 already supports multidesktop. (By press Ctrl+Windows+D to create a new one, or by click the button on the taskbar). what if we can appoint another file as new desktop when createing one?
